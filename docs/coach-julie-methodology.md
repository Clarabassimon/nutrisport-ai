# Méthodologie Coach Julie CrossFit — Référence

## Source: BesoinsnutritonnelsMacronutriments.pdf

### Méthode 1 — CrossFit (formule en livres)
- Poids en LBS = poids (kg) × 2.2
- Niveau d'activité (NA) :
  - 0.8 = Inactif/sédentaire
  - 0.9 = Activité modérée (sport 3x/semaine)
  - 1.0 = Actif (sport 4-5x/semaine)
  - 1.2 = Très actif (entraînement quotidien ou plusieurs fois/jour)
- **Protéines** = PC(lbs) × NA (en grammes)
- **Glucides** = Protéines(g) × 1.33
- **Lipides** = Protéines(g) × 4/9
- Ratio cible : 40% Glucides / 30% Protéines / 30% Lipides

### Méthode 2 — Classique (par kg de poids de corps)
- **Glucides** : 4 à 7g/kg PDC/jour
  - 4g/kg pour 2-3 séances/semaine
  - 7g/kg pour pratique très intense (5-6x/semaine)
- **Protéines** : 1.6 à 2.2g/kg PDC/jour
  - 1.6g/kg activité modérée à importante
- **Lipides** : 0.8 à 1g/kg PDC/jour

### Calories
- 1g protéines = 4 kcal
- 1g glucides = 4 kcal
- 1g lipides = 9 kcal

### Exemple Mme L (65kg, 165cm, 3x/semaine)
- Méthode CrossFit : 129g P / 172g G / 57g L = ~1720 kcal
- Méthode classique : 104g P / 260g G / 52g L = ~1924 kcal

### Principe fondamental
> "La seule façon de connaître vos calories de maintien quotidiennes est de compter avec précision vos quantités ingérées sur une ou deux semaines avec un poids stable."

## Source: GuidenutritionJulie.pdf

### Principes CrossFit Nutrition
- Manger des aliments de qualité, non transformés
- Privilégier les aliments bruts (cuisiner soi-même)
- Éviter les sucres ajoutés, différencier des glucides complexes

### Mise en pratique — Exemple assiette composée
- 120g blanc de poulet (132kcal, P=30g, G=0, L=2g)
- 60g riz pesé CRU (210kcal, P=5g, G=43g, L=0.5g)
- 150g haricots verts (47kcal, P=3g, G=11g, L=0)
- 1CS huile d'olive (124kcal, P=0, G=0, L=14g)
- **Total : 511 kcal (P=38g, G=54g, L=17g) = ratio 40/30/30 parfait**

### Règle importante : PESER CRU
- Les aliments doivent être pesés CRU
- Indiquer l'équivalent cuit à côté (approximatif)

## Source: Complémentsalimentaires012024.pdf

### Compléments recommandés (CrossFit)
1. **Protéine en poudre** : WHEY ISOLATE (préférer à concentrate), ou vegan si intolérance lactose
   - Vérifier : 3g leucine/portion, éviter acésulfame K, aspartame
2. **Oméga 3** : 1200mg EPA + 600mg DHA minimum/portion
   - Marques : Nutripure, Nutrimuscle
3. **Vitamine D** : 400 UI/j (officiel), 2000-4000 UI/j en hiver pour sportif
4. **Créatine** : 3-5g/jour, monohydrate (Creapure)
   - Bénéfices : prise de masse, force, puissance, récupération
5. **Glycine** : digestion, sommeil, masse musculaire, articulations
6. **Collagène** : type 1&3 (peau, muscles, os), type 2 (cartilage)
7. **Ashwagandha** : adaptogène, stress, énergie, prise de muscle

### 5. Coefficients de cuisson (cru → cuit)
- Riz : ×2.5 (60g cru → 150g cuit)
- Pâtes : ×2.2
- Viandes : ×0.75 (perte d'eau)
- Légumineuses : ×2.5
- Légumes : ×0.85
